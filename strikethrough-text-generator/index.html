What Is a Strikethrough Text Generator?

Strikethrough text generators apply the Unicode combining strikethrough character (U+0336) to each letter in your text. This combining character sits directly through the middle of any base character it follows, creating the visual effect of a horizontal line drawn through the text — similar to the HTML <s> or CSS text-decoration: line-through, but as plain text that works anywhere.

Several line overlay variants are available beyond simple strikethrough: underline (U+0332), overline (U+0305), double strikethrough (U+0336 + U+0335), wavy underline (U+0330), and double underline (U+0333). Each uses a different Unicode combining character from the Combining Diacritical Marks block.

Expert Tips for Lines & Overlays Text

Get better results with these pro techniques.

1

Short phrases only

Strikethrough text is legible for short phrases (3–6 words). Long paragraphs in strikethrough become genuinely hard to read — the line interferes with letter recognition.

2

Combine with normal text for comedy

The most effective strikethrough posts mix plain text with a strikethrough 'correction' — the contrast is what sells the joke. Don't strike through everything.

3

Try wavy underline for emphasis

The wavy underline style (U+0330) looks like a handwritten emphasis mark. It's subtler than strikethrough and works well for underlining key terms in posts.

4

Double strikethrough for maximum effect

For heavy visual impact, the double strikethrough style makes text look heavily redacted or censored — useful for dramatic effect in creative posts.

Frequently Asked Questions

Everything you need to know about the Strikethrough Text Generator.

How does Unicode strikethrough work?
Unicode combining character U+0336 (COMBINING LONG STROKE OVERLAY) attaches to the preceding character and renders as a horizontal line through it. Every letter in the word gets its own combining stroke.
Does strikethrough text work in WhatsApp?
Yes. Unicode combining strikethrough characters work in WhatsApp messages on both iOS and Android. Note that WhatsApp also supports its own Markdown ~tilde strikethrough~ which is simpler for basic use.
Can I use strikethrough in email subjects?
Email clients vary. Gmail renders Unicode combining characters in email subjects; Outlook may not. Test before using in marketing emails.
Is there a difference between strikethrough and redacted text?
Strikethrough (S̶t̶r̶i̶k̶e̶) uses Unicode combining characters and is still readable. True redaction uses solid blocks to hide text completely. For social media use, strikethrough is the standard choice.
